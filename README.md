## Project Description

The Movie Review Application is a mobile application developed using Flutter. The main purpose of this app is to allow users to browse movie information and submit their reviews for selected movies.

The application includes a simple login system, a home dashboard, movie collection page, review form, saved review database, and edit profile page. It is designed with a clean and user-friendly interface suitable for a mobile environment.

---

## Main Features

1. **Login Page**

   * Allows users to log in using the provided username and password.

2. **Home Dashboard**

   * Displays the main navigation menu of the application.

3. **Movie Collection**

   * Shows a list of available movies.

4. **Review Form**

   * Allows users to write and submit movie reviews.

5. **Saved Reviews**

   * Stores and displays submitted movie reviews.

6. **Edit Profile Page**

   * Allows users to edit their profile information.

7. **Database Integration**

   * Saves review details so that submitted reviews can be viewed later.

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/Arveengill/Movie-Review-Application-Flutter-App.git
```

### 2. Open the Project Folder

```bash
cd Movie-Review-Application-Flutter-App
```

### 3. Install Dependencies

```bash
flutter pub get
```

### 4. Run the Application

```bash
flutter run
```
