# Movie Review Application - Flutter App

This project is a mini project assignment for **UniKL** under the course:

**ISB26603 - Mobile and Ubiquitous Computing**

The purpose of this project is to develop a simple mobile application using Flutter. The application is based on a **Movie Review Application** theme, where users can log in, register, view movie information, check recent movie reviews, and navigate between different pages.

## Project Description

The Movie Review Application is designed to provide a clean and simple mobile interface for users to explore movie content. Users can view a list of movies, see movie images, read movie descriptions, and view recent reviews.

This project focuses on applying basic Flutter development concepts such as user interface design, navigation, form handling, validation, and proper project structure.

## Features

- Login page with username and password validation
- Registration page with form validation
- Home page with dashboard and recent movie reviews
- Media page displaying movie list using `Image.network`
- Movie detail page for viewing selected movie information
- Profile page with user/application information
- Form page for user input
- Bottom navigation bar
- Drawer navigation
- Clean and user-friendly UI design
